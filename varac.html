VarAC

VarAC is a real-time keyboard-to-keyboard chat application for amateur radio, developed by Irad Deutsch (4Z1AC). It uses the VARA modem for high-speed, reliable HF communications.

Overview

VarAC combines the speed and reliability of the VARA modem with a modern chat-style interface. Unlike FT8 or JS8Call which use timed transmission slots, VarAC operates in real-time like a telephone conversation.

Key Features

  • Real-time chat: No waiting for transmission windows
  • Beacon system: Automatic station announcements
  • File transfer: Send files, images, and forms
  • SNR reporting: Real-time signal quality display
  • CQ calling: Easy to call and find stations

Installation in EmComm-Tools

  1. Install and configure VARA HF first
  2. Download VarAC from https://varac-hamradio.com/
  3. Run installer: wine VarAC_setup.exe
  4. Configure VARA connection (localhost:8300)
  5. Enter your callsign and grid square

Operating VarAC

Calling CQ

  1. Select a clear frequency
  2. Click the "CQ" button
  3. Responding stations appear in the list
  4. Double-click to begin QSO
