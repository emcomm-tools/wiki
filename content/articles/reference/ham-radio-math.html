Ohm's Law & Power Formulas

The foundation of all electrical calculations. Memorize these!

Ohm's Law

V = I × R
I = V / R
R = V / I

Where: V = Voltage (volts), I = Current (amperes), R = Resistance (ohms)

Power Formulas

P = I × V (Power = Current × Voltage)
P = I² × R (Power = Current² × Resistance)
P = V² / R (Power = Voltage² / Resistance)

Where: P = Power (watts)

Example: A 12V radio draws 2A. What's the power consumption?
P = I × V = 2A × 12V = 24 watts

Decibel Calculations

Decibels (dB) express ratios logarithmically, making large ratios easier to work with.

Power Ratio

dB = 10 × log₁₀(P₂ / P₁)

Voltage Ratio

dB = 20 × log₁₀(V₂ / V₁)

Frequency & Wavelength

Basic Relationship

λ = c / f
Where: λ = wavelength, c = speed of light (3×10⁸ m/s), f = frequency

Practical Formulas

Wavelength (meters) = 300 / f(MHz)
Wavelength (feet) = 984 / f(MHz)
Example: What is the wavelength of 14.2 MHz (20m band)?
λ = 300 / 14.2 = 21.13 meters

Antenna Formulas

Half-Wave Dipole (total length)

Length (feet) = 468 / f(MHz)
Length (meters) = 143 / f(MHz)

Each leg is half this value.

Example: 40m dipole for 7.15 MHz
L = 468 / 7.15 = 65.45 feet total (32.7 feet each leg)

Quarter-Wave Vertical

Length (feet) = 234 / f(MHz)
Length (meters) = 71.5 / f(MHz)

5/8 Wave Vertical

Length (feet) = 585 / f(MHz)

Full Wave Loop

Length (feet) = 1005 / f(MHz)
Length (meters) = 306 / f(MHz)

Velocity Factor Adjustment

Actual Length = Calculated Length × VF

Common velocity factors:

  • Bare wire: 0.95
  • Insulated wire: 0.93-0.95
  • RG-58 coax: 0.66
  • RG-8X coax: 0.82
  • RG-213 coax: 0.66
  • Hardline: 0.81-0.88

Reactance & Impedance

Capacitive Reactance

Xc = 1 / (2πfC)

Where: Xc = capacitive reactance (ohms), f = frequency (Hz), C = capacitance (farads)

Inductive Reactance

XL = 2πfL

Where: XL = inductive reactance (ohms), f = frequency (Hz), L = inductance (henries)

Impedance

Z = √(R² + X²)
Z = √(R² + (XL - Xc)²)

Resonant Frequency

f = 1 / (2π√LC)

At resonance, XL = Xc and the circuit is purely resistive.

Example: Find resonant frequency with L=10µH and C=100pF
f = 1 / (2π√(10×10⁻⁶ × 100×10⁻¹²))
f = 1 / (2π√10⁻¹⁵) = 5.03 MHz

Q Factor

Q = XL / R = Xc / R (at resonance)
Q = f₀ / BW (where BW = bandwidth)

SWR & Transmission Lines

Standing Wave Ratio

SWR = Z_load / Z₀ (if Z_load > Z₀)
SWR = Z₀ / Z_load (if Z₀ > Z_load)

From Reflection Coefficient

Γ = (Z_load - Z₀) / (Z_load + Z₀)
SWR = (1 + |Γ|) / (1 - |Γ|)

Return Loss

Return Loss (dB) = -20 × log₁₀(|Γ|)
Return Loss (dB) = 20 × log₁₀((SWR+1)/(SWR-1))

Propagation Formulas

Free Space Path Loss

FSPL (dB) = 20×log₁₀(d) + 20×log₁₀(f) + 32.44

Where: d = distance (km), f = frequency (MHz)

Line of Sight Distance

d (miles) = √(2 × h)
d (km) = √(12.75 × h)

Where: h = antenna height in feet (miles formula) or meters (km formula)

Example: Antenna at 100 feet height
d = √(2 × 100) = 14.1 miles to radio horizon

Two Station Line of Sight

d (miles) = √(2 × h₁) + √(2 × h₂)

Maximum Usable Frequency (MUF)

MUF = Critical Frequency × sec(θ)

Where θ = angle of incidence to the ionosphere

Trigonometry for Antennas

Useful for calculating antenna angles, tower heights, and guy wire lengths.

sin(θ) = opposite / hypotenuse (SOH)
cos(θ) = adjacent / hypotenuse (CAH)
tan(θ) = opposite / adjacent (TOA)

Pythagorean Theorem

a² + b² = c²
Example: Guy wire from 50ft tower to anchor 30ft from base
c = √(50² + 30²) = √(2500 + 900) = √3400 = 58.3 feet
