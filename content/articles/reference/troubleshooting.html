Radio Won't Power On

ğŸ”´ Radio is Dead - No Display, No Lights

Quick Checks

  • Is the power switch ON? (Seriously, check it!)
  • Is the power cable connected at BOTH ends?
  • Is the fuse blown? Check inline fuse at battery.
  • Is the battery charged? Check voltage with meter.
  • Are Powerpole connectors fully seated?
  • Is polarity correct? (Red to +, Black to -)

Diagnosis Steps

  • Measure battery voltage: Should be 12-14V
  • Measure voltage AT THE RADIO power connector
  • Check for voltage drop in cable (should be < 0.5V)
  • Try different power cable if available
  • Try different battery/power source
Most common cause: Blown fuse or loose Powerpole connection
âš ï¸ Radio Powers On Then Shuts Off

Likely Causes

  • Undervoltage: Battery too weak, voltage drops on TX
  • Overcurrent: Power source can't supply enough current
  • Thermal: Radio overheating, thermal protection
  • Cable too thin: Voltage drop in undersized wires
Charge battery fully, use heavier gauge power cable, check connections

Can't Transmit

ğŸ”´ PTT Pressed But Radio Won't Transmit

Quick Checks

  • Is the antenna connected?
  • Is SWR protection kicking in? (Check SWR)
  • Is TX inhibit/lock enabled?
  • Are you on a frequency you're licensed for?
  • Is there a "TX" indicator when you key?
  • Is power set to something (not 0W)?
  • For FM: Is CTCSS/tone set if needed?
  • Is VOX enabled but not triggering?

For Repeater Operation

  • Is offset correct (+/âˆ’)?
  • Is CTCSS tone correct?
  • Are you in range of repeater?
  • Can you hear the repeater?
Most common: Wrong offset, wrong tone, or antenna disconnected
âš ï¸ Radio Transmits But No One Responds

Check These

  • Is your signal actually getting out? (Ask for signal report)
  • SWR OK? High SWR = weak signal
  • Are you on the right frequency?
  • For HF: Is propagation supporting your path?
  • Is the antenna resonant for this band?
  • Is power output what you expect?
Try a local repeater or known station to verify your signal

Can't Receive

ğŸ”´ No Audio / Dead Receiver

Quick Checks

  • Is volume turned up?
  • Is squelch set too high?
  • Is antenna connected?
  • Is external speaker plugged in but not connected?
  • Is headphone jack engaged?
  • Is AF gain (volume) up?
  • Is monitor/SQL button helping?

For FM

  • Is CTCSS/DCS tone set that shouldn't be?
  • Are you on correct frequency? (Check offset if repeater)

For SSB/HF

  • Is mode correct (USB vs LSB)?
  • Is RIT/XIT accidentally enabled?
  • Is bandwidth filter too narrow?
  • Is ATT (attenuator) or preamp wrong setting?
Open squelch to check for noise. If noise present, receiver works - check frequency/tone

High SWR

ğŸ”´ SWR Very High (>3:1) or Infinite

Immediate Checks

  • Is antenna connected? (Most common!)
  • Is coax connected at both ends?
  • Is coax damaged? (Look for kinks, cuts)
  • Are connectors tight?
  • Is connector center pin making contact?

If Connections Look Good

  • Is antenna designed for this band?
  • Is antenna tuned? (Did it work before?)
  • Did antenna move/fall/get damaged?
  • Is there water in coax or connector?
  • Test coax with dummy load - should be 1:1
Disconnect coax at antenna. Measure SWR into dummy load. If 1:1, problem is antenna. If still high, problem is coax.

Audio Problems

Reports of "Muffled" Audio

  • Mic too far from mouth
  • Mic gain too low
  • Compression too high
  • Dirty/blocked mic element

Reports of "Distorted" Audio

  • Mic gain too high
  • Speaking too loud/close
  • Over-deviation (FM)
  • ALC hitting hard (SSB)

Reports of "Hum" in Audio

  • Ground loop
  • Power supply issue
  • Shielding problem
  • Bad mic cable

No TX Audio

  • Mic not plugged in fully
  • Wrong mic type selected
  • Mic gain at zero
  • VOX not triggering

Digital Mode Issues

ğŸ”µ Digital Mode Not Decoding

Quick Checks

  • Is audio coming into the computer?
  • Check audio levels - not too high, not too low
  • Is software set to correct audio device?
  • Is radio in correct mode? (Usually USB for digital)
  • Is radio tuned to correct frequency?
  • Is computer time accurate? (Critical for FT8/JS8)

Audio Level Issues

  • Check waterfall - should see signals
  • If waterfall is blank - no audio input
  • If waterfall is solid color - audio too high/clipping
  • Adjust levels until signals visible but not clipping
For FT8/WSJT-X: Sync computer time with internet time server. Even 1-2 seconds off causes decode failures.
ğŸ”µ Digital Mode Not Transmitting

Check These

  • Is PTT/CAT control configured?
  • Is CAT cable connected?
  • Is correct COM port selected?
  • Does radio go into TX when software keys?
  • Is audio going to radio? (Check with headphones on line out)
  • Is radio DATA mode enabled (if required)?
  • Is VOX enabled (if using VOX for PTT)?

Power/ALC Issues

  • Is TX audio level appropriate?
  • Is ALC barely moving? (Should be minimal)
  • Is power output what you expect?
If using VOX: Make sure radio VOX is enabled and gain is set to trigger on digital audio

Interference (RFI)

âš ï¸ RF Getting Into Equipment

Symptoms

  • Computer resets when transmitting
  • Audio feedback/oscillation on TX
  • Erratic behavior of connected devices
  • USB disconnects during TX
  • "Hot" mic or chassis

Common Solutions

  • Add ferrite chokes to cables (USB, audio, power)
  • Use shorter cables
  • Improve station grounding
  • Lower power
  • Move antenna farther from station
  • Add common-mode choke to feedline
  • Check antenna system for RF on shield
Ferrite chokes (snap-on type) on ALL cables near the radio often fixes RFI issues
RFI Prevention: Add ferrites during initial setup, not after problems appear. A few dollars of ferrites saves hours of troubleshooting.
