Winlink is a worldwide radio email system that allows you to send and receive email via amateur radio. It's a critical tool for emergency communications when internet is unavailable.

Key Features

  • Email via RF - Send/receive email without internet
  • Store and Forward - Messages held until retrieved
  • Forms/Templates - Standardized EmComm forms (ICS, etc.)
  • Attachments - Small files, images (compressed)
  • Position Reporting - GPS location in messages
  • Peer-to-Peer - Direct station-to-station without gateway

Connection Modes

VARA HF

~8 kbps

Fastest HF mode

  • Worldwide range
  • Adaptive speed
  • Requires VARA license*

VARA FM

~25 kbps

VHF/UHF mode

  • Local/regional
  • Very fast
  • Requires VARA license*

ARDOP

~500 bps

Open source HF

  • Free, no license
  • Slower than VARA
  • Good backup

Packet

1200/9600 bps

AX.25 VHF/UHF

  • Free, proven
  • Widely available
  • Requires TNC

PACTOR

~5 kbps

Commercial HF

  • Very robust
  • Expensive hardware
  • SCS modem required

*VARA free version limited to ~175 bps. Full speed requires paid license.

Pat Client Setup

üì¶ What is Pat?

Pat is a cross-platform Winlink client that runs on Linux, Windows, macOS, and Raspberry Pi. It's the recommended client for EmComm-Tools.

  • Web-based interface (browser)
  • Command-line interface
  • Works with VARA, ARDOP, Packet, PACTOR
  • Free and open source

Installation (Linux/EmComm-Tools)

# Pat is pre-installed in EmComm-Tools # To verify installation: $ pat version # To start Pat web interface: $ pat http # Then open browser to: # http://localhost:8080

Initial Configuration

Pat configuration is stored in ~/.config/pat/config.json

Step 1: Set Your Callsign

$ pat configure

Or edit config.json directly:

{ "mycall": "VA2OPS", "secure_login_password": "YOUR_WINLINK_PASSWORD", "locator": "FN35", ... }

Step 2: Register with Winlink

Before using Winlink, register your callsign:

  1. Go to winlink.org
  2. Click "Create Account"
  3. Enter callsign and email
  4. Set a secure password
  5. Add password to Pat config

Step 3: Configure Audio Device

Pat needs to know which sound card to use for digital modes:

# List audio devices $ aplay -l $ arecord -l # In config.json, set ax25 or ardop sections: "ardop": { "addr": "localhost:8515", "arq_bandwidth": 500, ... }

VARA Modem Setup

VARA License: VARA HF and VARA FM are proprietary modems. The free version works but is speed-limited (~175 bps). Full speed requires purchasing a license from EA5HVK.

VARA HF Configuration

SettingRecommended ValueNotes
Soundcard Input Your interface (SignaLink, etc.) Must match radio connection
Soundcard Output Same as input (usually)
PTT Method CAT or VOX CAT preferred
Drive Level Start at 50% Adjust for ALC
Bandwidth 2300 Hz (default) 500 Hz for narrow

VARA + Pat Integration

In Pat's config.json:

"varahf": { "host": "localhost", "cmdPort": 8300, "dataPort": 8301 }, "varafm": { "host": "localhost", "cmdPort": 8300, "dataPort": 8301 }
Audio Levels: Set VARA drive level so your radio's ALC meter barely moves. Over-driving causes distortion and failed connections.

Basic Operation

Sending Email via Pat Web Interface

  1. Start Pat: pat http
  2. Open browser to http://localhost:8080
  3. Click Compose
  4. Enter recipient (callsign or email address)
  5. Write subject and message
  6. Click Post to Outbox
  7. Click Connect and select gateway
  8. Messages send/receive automatically

Addressing Messages

DestinationAddress FormatExample
Another ham CALLSIGN W1ABC
Internet email SMTP:address SMTP:john@example.com
Your Winlink email callsign@winlink.org va2ops@winlink.org
Message Size: Keep messages short! Large attachments take a long time over HF. Compress images. Avoid unnecessary attachments during emergencies.

Command Line Operation

# Connect to specific gateway via VARA HF $ pat connect varahf:///W3ADO?freq=7101000 # List messages in inbox $ pat inbox # Read a message $ pat read MESSAGE_ID # Compose from command line $ pat compose

EmComm Forms

üìã Winlink Standard Forms

Winlink includes standardized forms for emergency communications:

  • ICS-213 - General Message
  • HICS-213 - Hospital version
  • Red Cross - Disaster Welfare Inquiry
  • Radiogram - NTS format message
  • Check-In - Operator status
  • Situation Report - Field status
  • Resource Request - ICS resource forms

Using Forms in Pat

  1. Click Compose
  2. Click Select Template
  3. Choose form type
  4. Fill in form fields
  5. Post to Outbox and Connect

Peer-to-Peer (P2P) Mode

Winlink can operate station-to-station without a gateway:

  • Both stations need Pat + same modem (VARA, ARDOP)
  • One station initiates connection to other's callsign
  • Messages exchange directly
  • Useful when gateways unavailable
