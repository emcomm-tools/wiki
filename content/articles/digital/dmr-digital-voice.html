Digital Voice Overview

Digital voice modes encode your voice into digital data, transmit it, and decode it at the receiver. Advantages over analog FM:

  • Better audio quality - Consistent audio until signal is lost
  • No static/noise - Digital is either decoded or not
  • More efficient - Multiple conversations in same bandwidth (DMR)
  • Data capabilities - Text messaging, GPS, telemetry
  • Internet linking - Worldwide connectivity via reflectors/talkgroups

VOCODER (Voice Coder)

All three major digital voice modes use a vocoder chip to compress voice:

  • AMBE+2 - Used by DMR and D-STAR (DVSI chip)
  • IMBE - Used by P25 (public safety)
  • C4FM - Yaesu's implementation (also uses AMBE)

DMR (Digital Mobile Radio)

üìª DMR - Digital Mobile Radio

Originally a commercial standard (ETSI), now widely adopted by amateurs. Most popular digital voice mode due to affordable radios.

StandardETSI TS 102 361
Bandwidth12.5 kHz
Time Slots2 per channel
VocoderAMBE+2

DMR Key Concepts

TermWhat It Is
Time Slot (TS) DMR uses TDMA - 2 conversations on 1 frequency. TS1 and TS2.
Color Code (CC) Like CTCSS for DMR. Must match repeater (usually 1).
Talk Group (TG) Virtual "channel" - group of users. Like a room to join.
DMR ID Your unique ID number. Register at radioid.net
Private Call Direct call to specific DMR ID (like a phone call).
Codeplug Radio configuration file with all channels, TGs, contacts.

D-STAR

‚≠ê D-STAR (Digital Smart Technologies for Amateur Radio)

Developed by JARL (Japan), first amateur-specific digital voice mode. Primarily Icom equipment.

DeveloperJARL/Icom
Data Rate4.8 kbps voice
Bandwidth6.25 kHz
VocoderAMBE

D-STAR Key Concepts

TermWhat It Is
MY Call Your callsign (up to 8 chars)
UR Call Destination - CQCQCQ for general, or specific call/reflector
RPT1 Repeater you're accessing (with module letter)
RPT2 Gateway callsign (usually repeater call + G)
Reflector Linking system - REF, XRF, DCS, XLX
Module A, B, or C - different ports on a repeater

Wires-X

Wires-X is Yaesu's internet linking system for Fusion:

  • Rooms - Like talk groups, places to gather
  • Nodes - Repeaters or hotspots on the network
  • Connect via radio - Search and connect from your HT
  • News stations - Store-and-forward messaging
Fusion Simplicity: Unlike DMR, you don't need a "codeplug" or ID registration. Just program the frequency and go - the radio figures out the rest!

Getting Started

DMR Quick Start

  1. Get a DMR ID - Register at radioid.net
  2. Get a radio - TYT, Anytone, Radioddity popular choices
  3. Get/create a codeplug - Configuration for your area
  4. Program the radio - Use CPS (Customer Programming Software)
  5. Find local repeaters - Or set up a hotspot
  6. Start on TG 9990 - Parrot/echo to test your setup
  7. Try TG 91 or local TGs - Make contacts!

D-STAR Quick Start

  1. Get a D-STAR radio - Icom ID-52, ID-51, IC-705, etc.
  2. Program MY callsign - Your call in MY field
  3. Find local repeater - Program RPT1 and RPT2
  4. Set UR to CQCQCQ - For general calling
  5. Register at gateway - Some require registration
  6. Connect to reflector - Set UR to link command

Fusion Quick Start

  1. Get a Fusion radio - Yaesu FT-70D, FT-5D, FTM-300, etc.
  2. Program repeater frequency - Like analog FM
  3. Set mode to AUTO - AMS handles the rest
  4. Key up and talk! - That's it, really
  5. For Wires-X - Use DX button to search rooms
Listen First! Digital voice has the same courtesy rules as analog. Listen before transmitting, don't "kerchunk" repeaters, and be patient - there's often a delay on linked systems.
