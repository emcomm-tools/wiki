Yaesu Radios

FT-991A

SettingValue
Hamlib Model1035
Baud Rate38400
PTT MethodRIG (CAT)

Menu Settings

  • 031 CAT RATE = 38400bps
  • 032 CAT TOT = 100ms
  • 033 CAT RTS = DISABLE
  • 062 DATA MODE = OTHERS
  • 064 OTHER DISP (SSB) = 1500Hz
  • 065 OTHER SHIFT (SSB) = 1500Hz
  • 066 DATA LCUT FREQ = OFF
  • 068 DATA HCUT FREQ = OFF
  • 070 DATA IN SELECT = REAR
  • 071 DATA PTT SELECT = DAKY
  • 072 DATA PORT SELECT = USB
  • 073 DATA OUT LEVEL = 50
  • 076 FM PKT PTT SELECT = DAKY
  • 077 FM PKT PORT SELECT = USB
  • 078 FM PKT TX GAIN = 50
  • 079 FM PKT MODE = 1200

Operating Notes

  • For HF digital operation: Mode = DATA-USB
  • For HF digital operation: WDH = 3000Hz
  • For FM packet operation: Mode = DATA-FM

FTX-1 (No CAT)

SettingValue
Hamlib Model1 (Dummy)
Baud Rate38400
PTT MethodRTS (PTT only)

Operating Notes

  • Tune radio to desired frequency manually
  • Long press [MODE], then set accordingly for HF or VHF/UHF
  • For HF: Select 'DATA-U', then long press 'PRESET' and select 'FT8'
  • For VHF/UHF FM: Select 'FM', then long press 'PRESET' and select 'FT8'
  • The 'PRESET' button should be highlighted in blue when configured

G90 (with DigiRig)

SettingValue
Hamlib Model3088
Baud Rate19200
PTT MethodRTS

Radio Settings

  • AGC = OFF (AGC--)
  • MODE = U-D
  • RF-GAIN = 50%
  • AUX IN = 10
  • AUX OUT = 13

How to Set

  • AGC: Short press [AGC]
  • Mode: Press Mode arrow keys on top of head unit
  • RF Gain: Long press [AGC], use VFO knob
  • AUX IN: Long press [FUNC], Menu 5, use VFO
  • AUX OUT: Long press [FUNC], Menu 6, use VFO

Other Radios

BG2FX FX-4CR

SettingValue
Hamlib Model2031
Baud Rate115200
PTT MethodRIG (CAT)
Special Configdata_bits=8, serial_handshake=None, stop_bits=1, rts_state=OFF

Configuration Notes

  • Short press [PWR], then rotate AF knob until POW = 5.0
  • Toggle [SSB] button until both USB and DIG appear
  • Short press [M. V] until VFOA is displayed
  • Long press [MENU], rotate AF to BLUETOOTH, rotate TUNE and set to 0
  • Set radio volume to 50
  • Set mic gain to 50
  • Set alsamixer PCM to 21
  • Set alsamixer Mic to 55
  • Set alsamixer AGC to mm

Field Notes

  • Tested with FX-4CR firmware version: 1.7.3 (2025/09/01)
  • Important: You must force the RTS control line to low for audio output to work with PTT
  • This is handled with rts_state=OFF using the --set-conf argument of rigctld
