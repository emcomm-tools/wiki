Digirig Mobile Setup Guide

The Digirig Mobile is a compact, all-in-one digital mode interface combining sound card, CAT control, and PTT in a single USB device. This guide covers setup in EmComm-Tools.

Overview

The Digirig integrates:

  • USB Sound Card: Audio in/out for digital modes
  • Serial Port: CAT control and PTT
  • Compact Size: Fits in your palm

EmComm-Tools PnP Support

EmComm-Tools includes automatic detection:

  • Digirig appears as /dev/digirig
  • Audio device: "USB Audio CODEC"
  • No driver installation needed

Connections

To Computer

  • USB-C (Digirig side)
  • USB-A or USB-C (computer side)

To Radio

Digirig uses a 3.5mm TRRS connector:

RingSignal
TipAudio Out (to radio)
Ring 1Audio In (from radio)
Ring 2PTT (ground to key)
SleeveGround

Use the correct cable for your radio model.

Supported Radios

Digirig offers cables for:

  • Icom: IC-705, IC-7300, IC-706 series
  • Yaesu: FT-891, FT-857, FT-817/818, FT-991
  • Kenwood: TS-480, TH-D74
  • Elecraft: KX2, KX3
  • Xiegu: G90, X6100
  • HTs: Most via speaker/mic jacks

# Find serial port

Audio Level Adjustment

Transmit Level

  1. Open alsamixer
  2. Press F6, select USB Audio CODEC
  3. Adjust playback to ~70%
  4. Key transmitter, adjust until ALC barely moves

Receive Level

  1. In application waterfall, signals should peak 30-50 dB
  2. Adjust radio AF/volume if too high or low
  3. Or adjust capture in alsamixer

Testing

Test Audio Path

# Record 5 seconds
arecord -D plughw:X,0 -d 5 -f cd test.wav

# Play back
aplay test.wav

Test CAT Control

# Using rigctl
rigctl -m 3085 -r /dev/digirig -s 9600 f

Troubleshooting

Device Not Detected

  • Try different USB port
  • Check dmesg | tail -20
  • Use data-capable USB cable

No Audio

  • Verify cable fully seated
  • Check radio mode (USB/DATA)
  • Test with alsamixer

PTT Not Working

  • Verify CAT settings match radio
  • Try RTS instead of CAT
  • Check serial permissions
