JS8Call Quick Start Guide

JS8Call is a weak-signal digital mode designed for keyboard-to-keyboard messaging. It's based on FT8 but optimized for conversational QSOs and store-and-forward messaging.

Starting JS8Call in EmComm-Tools

Launch JS8Call from the applications menu or click the JS8Call icon on the desktop. EmComm-Tools is configured with PnP (Plug and Play) support for Digirig and similar interfaces.

Initial Configuration

Station Information

  1. Go to File → Settings → General
  2. Enter your callsign
  3. Enter your grid square (4 or 6 character Maidenhead locator)
  4. Set your station info message (optional)

Audio Setup

  1. Go to File → Settings → Audio
  2. For Digirig: Select "USB Audio CODEC" for both input and output
  3. Leave sample rate at 48000

Radio/CAT Control

  1. Go to File → Settings → Radio
  2. Select your rig from the dropdown
  3. Set the serial port (usually /dev/ttyUSB0 or /dev/digirig)
  4. Configure baud rate per your radio's settings
  5. Set PTT method (CAT, RTS, or VOX)
  6. Click "Test CAT" to verify connection

Common Commands

JS8Call uses special commands prefixed with specific characters:

CommandDescription
CALLSIGN HEARTBEATRequest heartbeat from station
CALLSIGN SNR?Request signal report
CALLSIGN INFO?Request station info
CALLSIGN GRID?Request grid square
@ALLCALL CQCall CQ to all stations
@HB HEARTBEATSend heartbeat to network
@APRSIS MSG messageSend APRS message (if gateway available)

Making a Contact

  1. Ensure your audio levels are set (use the tune button to check)
  2. Select a clear frequency offset in the waterfall
  3. Type @ALLCALL CQ or double-click a calling station
  4. When someone responds, their callsign appears in the activity window
  5. Double-click their callsign to start a directed QSO
  6. Type your message and click Send or press Enter
  7. End with 73 and SK (silent key = end of contact)

EmComm Features

JS8Call includes several features useful for emergency communications:

  • Store and Forward: Messages can be relayed through other stations
  • Automatic Relay: Enable to act as a message relay station
  • Heartbeat Network: Provides network status awareness
  • APRS Gateway: Send position and messages to APRS network
  • Group Calls: Use @GROUP for directed group messaging

Troubleshooting

No Decode

  • Check audio input levels (should peak around 30-50 dB)
  • Verify correct audio device selected
  • Confirm radio is in USB mode
  • Check that frequency is correct (dial + audio offset)

No Transmit

  • Test CAT control in settings
  • Verify PTT method matches your setup
  • Check audio output device
  • Confirm TX audio levels aren't causing ALC action
