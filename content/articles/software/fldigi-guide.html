Fldigi

Fldigi (Fast Light Digital) is a comprehensive digital mode application supporting over 80 modes including PSK31, RTTY, MFSK, Olivia, and many more. It's the Swiss Army knife of amateur radio digital modes.

Overview

Fldigi is free, open-source software that turns your computer into a multi-mode digital transceiver. With waterfall display, macro support, and logging, it's ideal for casual QSOs, contesting, and emergency communications.

Key Features

  • 80+ modes: PSK, RTTY, Olivia, MT63, Thor, etc.
  • Waterfall display: Visual tuning
  • Macro system: Automate common exchanges
  • CAT control: Integrated rig control
  • Logging: Built-in contest logger
  • RSID: Automatic mode detection

Companion Applications

  • Flmsg: NBEMS forms and radiograms
  • Flamp: File transfer
  • Flwrap: File wrapper/compression
  • Flarq: ARQ file transfer

Configuration

Audio Setup

  1. Configure → Sound Card
  2. Select PortAudio
  3. Input: Your USB interface
  4. Output: Same interface

Rig Control

  1. Configure → Rig Control
  2. Select Hamlib or RigCAT
  3. Choose rig model
  4. Set serial port and parameters
  5. Click Initialize

Operator Info

  1. Configure → Operator
  2. Enter callsign, name, QTH
  3. Set grid locator

Operating

Tuning

  1. Watch the waterfall for signals
  2. Click on a signal to tune
  3. Decoded text appears in receive pane
  4. AFC (Auto Frequency Control) tracks drift

Transmitting

  1. Type in transmit pane
  2. Press TX button or F1 macro
  3. Text transmits character-by-character
  4. Press RX or Escape to stop

Using Macros

Function keys trigger customizable macros:

KeyTypical Use

F573, end QSO

NBEMS (Narrow Band Emergency Messaging)

Fldigi is central to NBEMS - a system for sending structured messages and forms over radio:

  • Flmsg: Create ICS forms, radiograms
  • Flamp: Transfer files with ARQ
  • Recommended mode: MT63-1000 or Olivia

RSID (Reed-Solomon ID)

Fldigi can auto-detect modes using RSID:

  • Enable RSID receive in Configure → IDs
  • Fldigi switches modes automatically
  • Enable RSID transmit to help others

Troubleshooting

No Audio In

  • Check sound card selection
  • Verify levels in alsamixer
  • Test with arecord command

Garbled Decode

  • Wrong mode selected
  • AFC not locked
  • Signal too weak or overdriven

PTT Issues

  • Check rig control configuration
  • Verify serial port permissions
  • Try different PTT method
