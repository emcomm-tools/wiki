BPQ/LinBPQ Commands Reference

Complete command reference for BPQ32 and LinBPQ packet node and BBS software. LinBPQ is included in EmComm-Tools for running packet radio nodes, BBS systems, and chat servers over AX.25.

Node User Level Commands

These commands are available to all users connecting to the node.

CommandDescription
? Displays list of commonly used node commands, starting with sysop-defined applications, then standard ones.
Attach [P] Get exclusive control of port before connecting (for single-user ports like WINMOR or Pactor).
BBS [Sysop defined] Access the BBS and Mail system.
Bye or Quit Disconnect from Node.
CHAT [Sysop defined] Access the Chat system.
Connect Initiate connection to another node or AX.25 user.
C P CALL|ALIAS [via digi1 [digi2...]] [S]
P = port number, S = STAY connected to current node when next closes.
Use ! prefix to force Level 2 connect: C 1 !CALL
CQ Send beacon to address CQ. Only in LISTEN mode on one port.
IDLEtime Set idle session timeout for this session.
Info Display information about this Node from config file.
L4T1 Display/set Level 4 (NetROM) timeout for current session.
Links List currently active AX.25 sessions.
S=link state (2=connecting, 4=disconnecting, 5=connected)
P=port, T=link type (1=Uplink, 2=Downlink, 3=Node-Node)
V=AX.25 version (1 or 2)
LISten P1...Pn Enter Listen mode on specified port(s). Type LISTEN OFF to exit.
MHeard P [CALL] List stations heard recently on port P. Filter with partial call.
Variants: MHV (verbose), MHU (UTC), MHL (local time)
Nodes [CALL|ALIAS] [T] List all known NETROM/THENET/BPQ nodes.
RTT=Round Trip Time, FR=frames sent, B=BPQ node + hops
T option shows RTT and frame counts for all nodes.
NRR CALL Send NetROM Record Route message to specified node.
PAClen Display/set PACLEN value for node-generated messages.
Ports List available ports with frequency and baud rate.
RADio Used by Rig Control Interface.
Routes [r] List stations this node can hear. r r for detailed info.
Shows: port, quality, NODES entries, locked routes (!)
Stats Display system counters (uptime, buffers, L4 connects, frames, etc.)
S S = system/L3/4 stats only
S n = port stats starting at port n
SYSOP [Sysop defined] Talk to Sysop if available.
UNproto P CALL [Digis] Enter UNPROTO mode to send UI messages. Ctrl/Z to exit.
Users List stations currently using the node.
Host=internal, Circuit=node link, Uplink=user in, Downlink=user out
<--> = active, <~~> = setting up
Version Display Node software version number.

Node Sysop Level Commands

Sysop Authentication

When accessed remotely, enter PASSWORD followed by 5 characters from your password string corresponding to the random positions shown in the challenge prompt.

System Commands

CommandDescription
REBOOT Performs shutdown -r -f - reboots the machine.
RESTART Instructs node software to quit immediately.
ROUTES CALL P parms Update routes. Use ! flag to toggle locked routes.
SAVEMH Save and restore MH lists on shutdown/restart.
SAVENODES Save NODES and ROUTES tables for reload on restart.
SENDNODES Send NODES broadcast.
STARTCMS p Enable CMS access on Telnet port p.
STOPCMS p Disable CMS access on Telnet port p (simulate outage).
STOPPORT n Close port n (allows TNC reconfiguration).
STARTPORT n Reopen port n.
TELReconfig p ALL|USERS Reconfigure Telnet port. ALL=full reconfig, USERS=re-read users.
WL2KSYSOP [SET] Display or create Winlink 2000 SYSOP record.

Port Parameters

Display or update parameters for a specific port. Syntax: PARAM port# [=value]

ParameterDescription
BBSALIASBBS alias for this port.
DIGIFLAG=0|1|255Digipeat control: 0=don't digi, 1=digi all, 255=UI frames only.
DIGIPORT=NPort for digipeated frames. 0=same port.
FRACK=nnnnLevel 2 timeout in milliseconds.
FULLDUP=0|1Full duplex mode (KISS TNC).
L3ONLY=0|1If 1, prevents user downlink connects on this port.
MAXFRAME=nMaximum outstanding frames.
MAXUSERSMaximum users on this port.
PACLEN=nnnDefault maximum packet length.
PERSIST=nnnProbability to transmit value (1-255).
QUALITY=nnnDefault quality for nodes heard. 0=suppress L3/4 activity.
RESPTIME=nnnnLevel 2 delayed ACK timer in ms.
RETRIES=nLevel 2 maximum retry value.
SOFTDCD=0|1Use frame detection for DCD instead of modem signal.
TXDELAY=nnnTX keyup delay in milliseconds.
TXTAIL=nnnnTX tail timer in ms (KISS TNCs).
VALIDCALLS=CALLSWhitelist of allowed callsigns (up to 40, comma-separated).
XMITOFF port# [1]Disable transmitter on port. 0=enabled (default).

System Parameters

Display or update node-wide parameters.

ParameterDescription
BTINTERVALBeacon transmission interval.
FULLCTEXTFull connect text.
HIDENODESHide nodes with alias starting with #.
IDINTERVALStation ID interval in minutes.
L3TTLMaximum Level 3 hops.
L4DELAYLevel 4 delayed ACK timer.
L4TIMEOUTLevel 4 timeout.
L4RETRIESLevel 4 retries.
L4WINDOWLevel 4 window size.
LINKEDFLAGLinked flag setting.
MH p CLEARClear MHeard list for port p.
MINQUAL=NNNMinimum quality for NODES broadcast inclusion.
NODEIDLETIMEIdle timeout before link close (seconds).
NODESINTTime between NODES broadcasts (seconds).
OBSINITInitial obsolescence value.
OBSMINMinimum obsolescence for NODES broadcast.
T3Link validation timer (seconds).

BBS User Level Commands

General Commands

CommandDescription
? or HelpDisplay list of commands.
AAbort BBS output while paging.
ByeDisconnect from BBS.
CMSPASS passwordSet password for Winlink CMS access.
HomeEnter/modify Home BBS. Period (.) deletes current.
IDisplay BBS information.
I callsignInfo on callsign from White Pages (wildcards OK).
I@ bbsList users of BBS from WP database.
IH routeList users of hierarchical route from WP.
IZ zipList users by ZIP code from WP.
NEnter/modify your name (max 12 chars).
NODEExit BBS, return to Node.
OP numSet paging lines. 0=disable paging.
QEnter/modify QTH.
VDisplay BBS and Node version.
XToggle Expert Mode.
ZEnter/modify ZIP code.

List Messages (L Commands)

CommandDescription
LList new messages since last check.
L< callList messages FROM callsign.
L> callList messages TO callsign.
L@ bbsList messages VIA BBS.
L msg#-List all messages starting from msg#.
L msg#-msg#List messages between numbers.
LxList by status: N=New, Y=Read, H=Held, K=Killed, F=Forwarded, D=Delivered, $=BID.
LBList bulletins.
LCList all bulletin categories (TO fields).
LL nList last n messages.
LMList messages TO you.
LPList personal messages.
LRList in reverse order.
LTList NTS traffic messages.

Read Messages (R Commands)

CommandDescription
R msg# [msg#...]Read messages by number.
RMRead new messages to you.
READ nameRead file from Files area.

Send Messages (S Commands)

CommandDescription
S callSend private message.
SP callSend private message (explicit).
SB callSend bulletin.
S call @ BBSSend message to station at another BBS.
SC msg# call @ BBSCopy message to specified callsign.
SR msg#Reply to message.

Note: Terminate message with Ctrl/Z or /EX followed by Enter.

Kill Messages (K Commands)

CommandDescription
K msg#Delete message by number.
KMDelete all unread messages to you.

Other BBS Commands

CommandDescription
D numFlag NTS message as Delivered.
FILES or LISTFILESList files available for download.
POLLRMS ENABLE|DISABLEEnable/disable polling RMS for your messages.
POLLRMS SSID1 SSID2...Set SSIDs to poll (0=base call).
YAPP nameDownload file using YAPP protocol.

BBS Sysop Level Commands

CommandDescription
AUTHAuthenticate for remote SYSOP access (use BPQAUTH program).
EXPORT nnn filenameExport message nnn to file.
IMPORT filenameImport messages from file.
FWD bbscallDisplay forwarding settings for BBS.
FWD bbscall intervalSet forwarding interval.
FWD bbscall REV intervalSet reverse forwarding interval.
FWD bbscall +- flagsSet flags: EN(able), RE(verse Poll), SE(nd Immediately).
FWD bbscall NOWStart forwarding cycle immediately.
FWD QUEUEList BBSs with queued messages.
EDITUSER callDisplay user flags.
EDITUSER call FLAG...Set flags. Use - prefix to clear.
Flags: EXC(luded), EXP(ert), SYSOP, BBS, PMS, EMAIL, HOLD, RMS
KHKill all HELD messages.
K< callKill messages FROM callsign.
K> callKill messages TO callsign.
LHList HELD messages.
LKList KILLED messages.
UH ALLUnhold all messages.
UH msg# msg#...Unhold specific messages.
POLLRMS call ENABLE|DISABLESet RMS polling for other users.

Chat User Commands

Commands for the BPQ Chat system (all start with /).

CommandDescription
/AToggle alert on user join.
/AutoToggle automatic character set selection.
/BLeave Chat, return to Node.
/CToggle colour mode (Console/BPQTerminal only).
/EToggle echo.
/FForce all links to be made.
/J nameJoin or create topic (not case sensitive).
/KShow known nodes.
/KeepaliveToggle keepalive messages every 10 minutes.
/NEnter your name.
/PShow ports and links.
/QEnter your QTH.
/QUITLeave Chat and disconnect from Node.
/S call textSend private text to specific station.
/ShowNamesToggle showing name with call on messages.
/TShow topics.
/TimeToggle timestamp on messages.
/UShow users.
/UTF-8Toggle UTF-8 or 8-bit character set.
