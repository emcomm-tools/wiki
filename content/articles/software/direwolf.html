Direwolf

Direwolf is a software "soundcard" Terminal Node Controller (TNC) and APRS encoder/decoder, developed by John Langner (WB2OSZ). It turns any computer with a sound card into a packet radio modem.

Overview

Direwolf decodes and encodes AX.25 packet radio, APRS, and other digital modes using your computer's sound card. It replaces hardware TNCs and provides superior performance with modern error correction.

Key Features

  • Software TNC: No hardware TNC needed
  • APRS decode/encode: Full APRS support
  • KISS interface: Works with packet applications
  • AGW interface: Compatible with Windows apps
  • iGate capability: APRS-IS gateway
  • Digipeater: Packet relay station
  • Multiple modems: 1200 and 9600 baud

Configuration

Direwolf uses a configuration file, typically direwolf.conf:

Basic Configuration

# Audio device
ADEVICE plughw:1,0

# Your callsign
MYCALL VA2OPS-1

# Channel 0 settings (1200 baud VHF)
CHANNEL 0
MODEM 1200

# PTT control
PTT /dev/ttyUSB0 RTS

APRS iGate Configuration

# Enable iGate
IGSERVER noam.aprs2.net
IGLOGIN YOURCALL-10 12345

# Filter (receive only within 100km)
IGFILTER r/45.50/-73.50/100

Digipeater Configuration

# Enable digipeater
DIGIPEAT 0 0 ^WIDE[3-7]-[1-7]$|^TEST$ ^WIDE[12]-[12]$ TRACE

Audio Interface Setup

Find your audio device:

arecord -l   # List recording devices
aplay -l     # List playback devices

Example devices:

  • plughw:1,0 - USB sound card
  • plughw:CARD=Device,DEV=0 - By name

Running Direwolf

# Basic start
direwolf

# With specific config
direwolf -c /path/to/direwolf.conf

# With KISS port for other apps
direwolf -p

KISS Interface

Direwolf provides a KISS TCP port for other applications:

  • Default port: 8001
  • Use with: YAAC, Xastir, Pat, linBPQ

Example: Connect YAAC

  1. Start Direwolf with -p flag
  2. In YAAC: Add port → KISS-over-TCP
  3. Host: localhost, Port: 8001

AGW Interface

For Windows applications via Wine:

  • Default port: 8000
  • Compatible with: UI-View, APRSISCE

Troubleshooting

No Audio

  • Check ADEVICE setting matches your hardware
  • Run alsamixer to check levels
  • Test with arecord and aplay

No Decode

  • Check audio levels (not clipping)
  • Verify frequency is correct
  • Try adjusting squelch on radio

PTT Not Working

  • Check serial port permissions
  • Verify RTS/DTR setting
  • Test with stty
