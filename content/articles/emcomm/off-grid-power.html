Essential Formulas

Power (W) = Voltage (V) √ó Current (A)

Watt-hours (Wh) = Volts (V) √ó Amp-hours (Ah)

Amp-hours (Ah) = Watt-hours (Wh) √∑ Volts (V)
Example: A 12V 20Ah battery
Energy = 12V √ó 20Ah = 240 Wh

Battery Runtime Calculator

The Basic Formula

Runtime (hours) = Battery Capacity (Ah) √ó DoD √ó Efficiency √∑ Load (A)

Where:

  • DoD (Depth of Discharge) - How much you should drain the battery (0.5 for lead-acid, 0.8 for LiFePO4)
  • Efficiency - Account for losses (typically 0.85-0.95)
  • Load - Average current draw in amps

Simplified Formula (Conservative)

Runtime (hours) = Battery Ah √ó 0.5 √∑ Average Amps

This assumes 50% discharge (safe for lead-acid) and accounts for efficiency losses.

üîã Quick Runtime Examples

Setup: 20Ah LiFePO4 battery, IC-705 at 5W (average 1.5A with RX)

Runtime = 20Ah √ó 0.8 √∑ 1.5A = 10.6 hours

Setup: 100Ah lead-acid, IC-7300 at 100W (TX 20A, RX 1A, 20% TX duty)

Average current = (20A √ó 0.2) + (1A √ó 0.8) = 4A + 0.8A = 4.8A

Runtime = 100Ah √ó 0.5 √∑ 4.8A = 10.4 hours

Solar Panel Sizing

Basic Solar Formula

Panel Watts = Daily Wh Needed √∑ Sun Hours √∑ 0.7

The 0.7 factor accounts for system losses (charge controller, wiring, angle, clouds, etc.)

‚òÄÔ∏è Solar Sizing Example

Need: Run IC-7300 for 8 hours at 20% duty cycle

Average draw = (20A √ó 0.2) + (1A √ó 0.8) = 4.8A

Daily Wh = 4.8A √ó 12V √ó 8h = 461 Wh

Assume 5 peak sun hours (average US)

Panel = 461 Wh √∑ 5h √∑ 0.7 = 132W panel minimum

Recommendation: 150-200W panel for margin

Charge Controllers

PWM vs MPPT

TypeEfficiencyCostBest For
PWM 75-80% $ Small panels, matched voltage (12V panel ‚Üí 12V battery)
MPPT 94-98% $$-$$$ Larger systems, mismatched voltages, max efficiency
Rule of Thumb: For systems over 100W, MPPT pays for itself in efficiency gains. For small QRP setups, PWM is fine.

Controller Sizing

Controller Amps = Panel Watts √∑ Battery Voltage √ó 1.25
Example: 200W panel into 12V battery
Controller = 200W √∑ 12V √ó 1.25 = 21A minimum (get 30A)

Fuse Sizing

Fuse = Maximum Current √ó 1.25

Round up to next standard size (5A, 10A, 15A, 20A, 25A, 30A, etc.)

Always Fuse! Install fuse as close to battery positive terminal as possible. This protects against fire if wire shorts to ground.

Anderson Powerpole Standard

The ham radio standard for 12V DC connections:

  • 15A/30A/45A - Common sizes
  • Red on top (or left when horizontal) - Positive
  • Black on bottom (or right) - Negative
  • Tongue on red facing up, tongue on black facing down
