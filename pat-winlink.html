Pat Winlink

Pat is a cross-platform Winlink client for Linux, macOS, and Windows, developed by Martin Hebnes Pedersen (LA5NTA). It provides email over radio via the Winlink network.

Overview

Pat is the premier Winlink client for Linux users. It supports multiple connection methods including VARA, ARDOP, Packet, Pactor, and Telnet, with both command-line and web-based interfaces.

Key Features

  • Multiple modems: VARA, ARDOP, Packet, Pactor
  • Web interface: Easy browser-based operation
  • CLI interface: Scriptable command line
  • Forms support: ICS and Winlink forms
  • Position reporting: GPS integration
  • P2P messaging: Direct station-to-station

Installation

Pat is pre-installed in EmComm-Tools. To verify:

pat version

Initial Configuration

  1. Run configuration wizard: pat configure
  2. Or edit config directly: ~/.config/pat/config.json

Essential Settings

{
  "mycall": "VA2OPS",
  "secure_login_password": "your-winlink-password",
  "locator": "FN35",
  "http_addr": "localhost:8080"
}

Connection Methods

VARA HF

# In config.json, add VARA section:
"vara": {
  "host": "localhost",
  "cmdPort": 8300,
  "dataPort": 8301
}

# Connect to gateway
pat connect vara:///WL2K-GATEWAY

ARDOP

# Start ARDOP modem first
ardopcf 8515 plughw:1,0 plughw:1,0

# Connect
pat connect ardop:///WL2K-GATEWAY

Packet (via Direwolf)

# Start Direwolf first
# Then connect via AX.25
pat connect ax25:///WL2K-GATEWAY

Telnet (Internet)

# For testing without radio
pat connect telnet

Web Interface

  1. Start Pat HTTP server: pat http
  2. Open browser: http://localhost:8080
  3. Compose, read, and manage email via web UI

Command Line Usage

# List available gateways
pat rmslist

# Connect to specific gateway
pat connect vara:///K4RGN

# Compose new message
pat compose

# Read inbox
pat read

# Interactive mode
pat interactive

Finding Gateways

# List HF gateways within range
pat rmslist -m vara -d 5000

# List VHF gateways nearby  
pat rmslist -m packet -d 100

Forms Support

Pat supports Winlink standard forms:

  • ICS-213 General Message
  • Radiogram
  • Red Cross Safe and Well
  • Weather reports

Access forms through the web interface Compose menu.

Position Reporting

# Configure GPS in config.json
"gpsd": {
  "enable_http": true,
  "addr": "localhost:2947"
}

# Or set fixed position
"locator": "FN35at"

Aliases

Create shortcuts for common gateways:

# In config.json
"connect_aliases": {
  "hf40": "vara:///WL2K@7101500",
  "local": "packet:///WL2K-VHF"
}

# Then use
pat connect hf40

Troubleshooting

VARA Connection Failed

  • Ensure VARA HF is running
  • Check ports (8300/8301)
  • Verify audio setup in VARA

Authentication Error

  • Verify Winlink password in config
  • Register at winlink.org if needed
  • Check callsign matches registration

No Gateways Found

  • Check internet connection (for list)
  • Verify frequency/band selection
  • Try different distance setting

EmComm Usage

Pat is essential for emergency communications:

  • Email when internet is down
  • ICS forms for official traffic
  • Position reports for tracking
  • Works with any Winlink gateway
